# TikDown.click

A simple tiktok video downloader using vue.js

## Installation 

### Client

Chnage directory to '/client' folder.
```sh
cd client
```
Install dependencies.
```sh
yarn install
```
Finally, Run the application.
```sh
yarn run dev
```

### Server

Change directory to '/server' folder.
```sh
cd server
```
Install dependencies 
```sh
npm install
```
Finally, Run the server
```sh
npm run start
```
